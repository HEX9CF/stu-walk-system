ChaoUser 类实现了一个简单的用户数据模型。它表示系统中的一个用户，包含了用户的基本信息，如用户 ID、用户名和密码。

主要功能：
1.属性（实例变量）：
  id：用户的唯一标识符，类型为 Integer。
  username：用户的用户名，类型为 String。
  password：用户的密码，类型为 String。
2.Getter 和 Setter 方法：提供了标准的访问器方法：
  getId()：返回用户的 ID。
  setId(Integer id)：设置用户的 ID。
  getUsername()：返回用户的用户名。
  setUsername(String username)：设置用户的用户名。
  getPassword()：返回用户的密码。
  setPassword(String password)：设置用户的密码。
  toString() 方法：重写了 Object 类的 toString() 方法，返回一个表示用户对象的字符串形式。这个字符串包括用户的 ID、用户名和密码，格式如下：
    ChaoUser{id=1, username='user1', password='password123'}

主要用途：
  1.该类可以用于表示系统中的用户对象，便于存储、传递和操作用户的基本信息。
  2.可作为数据库操作、用户验证、用户管理等功能的基础数据结构。
  3.通过 getId(), getUsername(), getPassword() 等方法，系统可以获取用户的具体信息。
  4.通过 toString() 方法，可以方便地输出用户的基本信息，通常用于调试或日志记录。
